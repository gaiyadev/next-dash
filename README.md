# Next-Dash

Next-Dash is a free and open source NextJs dashboard built with Material UI.

## Installation

Use the package manager to install next-dash.

```bash
npm install next-dash
```

## Usage

Clone and repository and follow the following steps

```bash
cd next-dash
yarn install
yarn dev
```
